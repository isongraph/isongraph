# Changelog

## 0.1.0

- Initial release.
- Deterministic seeded force layout — exact mirror of `ison_graph.viz.compute_layout`.
- SVG rendering with type-colored nodes, degree sizing, legend, arrowheads, tooltips.
- Live physics: draggable nodes with animated relaxation, plus Reset layout / Fit.
- Live re-render on file edit. Zero runtime dependencies, strict webview CSP.
